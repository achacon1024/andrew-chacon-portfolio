# Artifact 1 – Software Design & Engineering Enhancement

## Overview
This artifact is an enhanced version of the Grazioso Salvare Animal Shelter Dashboard originally developed in CS 340. The application is built using Python, Dash, and MongoDB.

## Enhancement Focus
The software design and engineering enhancement focused on:

- Improving modularity by separating database logic into a dedicated CRUD module (`animal_shelter_crud.py`)
- Enhancing error handling for database operations
- Improving overall code organization and maintainability
- Strengthening connection security through authenticated MongoDB access

## Technologies Used
- Python
- Dash (Plotly)
- MongoDB
- PyMongo

## How to Run
1. Ensure MongoDB is running locally.
2. Update database credentials inside the dashboard file if needed.
3. Run the Dash application file.
4. Open the local server URL provided in the terminal.

This enhancement demonstrates improved system structure, modular architecture, and maintainable design practices.
