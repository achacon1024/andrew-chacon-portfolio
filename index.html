Introduction

This ePortfolio represents the culmination of my work throughout the Computer Science program at Southern New Hampshire University. The featured artifact is a full-stack Animal Shelter Dashboard application built using Python, Dash, and MongoDB. For this capstone, I revisited and enhanced the original project across three outcome areas: software design and engineering, algorithms and data structures, and database implementation. These enhancements reflect my ability to critically analyze existing systems, improve efficiency and structure, and implement solutions grounded in sound technical principles. The final result demonstrates both practical development skills and a stronger understanding of system architecture and data management.
